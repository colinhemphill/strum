## Controlled input

To control your checkbox component with React, simply include a `value` boolean prop and a `onChange` function prop.

```tsx live name="controlled checkbox input" checked={false} changeCheckboxValue={(value) => {alert(value)}}
<Checkbox
  id="controlled-checkbox"
  label={`The value is ${checked}`}
  onChange={changeCheckboxValue}
  value={checked}
/>
```
