## Controlled input

To control your input with React, simply include a `value` string prop and a `onChange` function prop.

```tsx live name="controlled input" songTitle="Symphony No. 31" changeSongTitle={(e) => {alert(e.target.value)}}
<Stack>
  <Input
    id="input-text"
    label="Text input"
    onChange={changeSongTitle}
    type="text"
    value={songTitle}
  />
</Stack>
```
